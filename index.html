Web Mobile

This course aims to present a number of techniques and tools used to build mobile web applications. Multi Media Web

This course explains the basic concepts and principles of multimedia science: text, images, audio, video, colour, basic concepts of compression, and its various algorithms for text, images, audio, and video. At the end of this course, the student will be able to: (1) Recognize multimedia: texts, graphics, images, audio, and video. (2) Multimedia representation. (3) multimedia systems. (4) Understand the basic concepts of colors. (5) Understand basic concepts in the video (6) Identify basic concepts in animation (7) Identify basic concepts in compression algorithms

Data Mining

This is intended to be to provide students with the knowledge and skills that enable him to analyze large volumes of data and extract knowledge of them, and therefore make decisions based on this knowledge. The course mainly covers the following topics: (1) Collecting data from different sources in a data store. (2) Learn about the design and construction of data warehouses. (3) Learn about the various techniques of data mining: classification, regression, clustering, and association rules. (4) Acquiring new knowledge in the form of models. (5) Evaluation of acquired knowledge. (6) Identifying open research topics in this field.
